📁 Gestione Contenuti via GitHub

Poiché il sito è hostato su Cloudflare Pages, tutti i contenuti (auto e articoli) sono gestiti direttamente via GitHub nel tuo repository:

📦 exaauto/exaauto/data/
├── cars/ (annunci auto in JSON)
└── news/ (articoli in JSON)

Come funziona:

  • Aggiungi un'auto o un articolo nel form qui accanto
  • I dati vengono salvati come file JSON in GitHub
  • Cloudflare Pages legge automaticamente i file e li mostra sul sito
  • Non servono account esterni - tutto tramite GitHub!
📂 Visualizza Auto su GitHub 📂 Visualizza Articoli su GitHub

🔧 Configurazione Avanzata

Per modifiche più avanzate o per usare un CMS esterno, puoi configurare:

Opzione 1: Netlify CMS (alternativa)

Nonostante usi Cloudflare Pages, puoi comunque usare Netlify CMS per una gestione più facile. Contatta il supporto Netlify per la configurazione con Cloudflare Pages.

Opzione 2: Modifica diretta su GitHub

Accedi al tuo repository e modifica i file JSON nella cartella data/
